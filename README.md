**Dynamic Blog Hub**

A **Django-powered blog website** with seamless **API integration**, enabling users to **create, read, update, and delete blog posts** efficiently. This project is designed to be scalable, with a clean and modern interface.

## Features

✅ User authentication (Sign up, Login, Logout)  
✅ Create, Read, Update, and Delete (CRUD) blog posts  
✅ API integration using Django REST Framework (DRF)  
✅ Responsive UI with  CSS  
✅ Commenting system for user interactions  
✅ Database connectivity with **Supabase**  
✅ Deployment on **Render**
